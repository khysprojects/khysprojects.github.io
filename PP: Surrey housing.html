Surrey Housing Analysis

This project analysis aims to explore housing trends across local authorities in Surrey using publicly available datasets. It focuses on key metrics such as rental prices, housing tenure, affordability, and homelessness to better understand housing pressure and market differences. The project examines these through data cleaning, data processing, modelling, and clear visualisation, allowing decisions to be made with confidence.
